# Práctica 4 - Soluciones en la Nube

## Pasos para instalar:
1. Clonar este repositorio.
2. Configurar la base de datos en AWS RDS (PostgreSQL).
3. Poner el endpoint de la base de datos en el archivo `app.py`.
4. Ejecutar con Docker:
   `docker-compose up --build`
   
## Tecnologías:
- Flask (Python)
- PostgreSQL (AWS RDS)
- Docker
